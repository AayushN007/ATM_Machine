# 🏧 ATM Machine Program (Python)

A simple **ATM Machine simulation program** built using Python.  
This project demonstrates basic programming concepts like **conditions, loops, user input, and variables**.

## 📌 Features

- Insert card simulation
- PIN verification
- Language selection
- Withdraw money
- Deposit money
- Balance enquiry
- Transaction status message
- Simple ATM workflow

## 🛠️ Technologies Used

- Python 3
